# pizero_cam_case

Simple case(s) for mounting a Raspberry Pi Zero with a camera to the wall - used in laundry room.

Doesn't fit quite perfectly, but it does the job.

Anybody with access to UoE Main Library can make use of the uCreate Makerspace on the first floor -- get it 3D printed for free there! Using the Prusa Minis, accessed through the print queue (https://www.ucreatestudio.is.ed.ac.uk/forms/3d_print_queue_submissions) is the most convenient option.

Now on github instead of our gitea instance.
